Model Architecture

IMChair 프로젝트에서는 전이학습(Transfer Learning)의 방식을 사용했다. 전이학습은 이미
대량의 데이터셋으로 학습된 모델의 지식을 활용하여, 새로운 모델을 빠르고 효율적으로
학습시키는 방법이다.

이미지 분류를 위해 널리 알려진 구글의 인셉션(Inception V3) 모델을 기반으로, 이미지의 다양한
요소와 형태를 인식하고 분석하는 모델을 구성했다.

딥러닝 모델을 훈련시키기 위해서는 대량의 라벨이 붙은 이미지 데이터가 필요하다.
이를 위해 IMChair 프로젝트에서는 효율적인 데이터 수집 방법을 모색하였고, 이 과정에서 생성형
인공지능, Stable Diffusion을 활용했다.

Stable Diffusion은 입력한 텍스트를 바탕으로 이미지를 생성할 수 있는 모델이다. 이를 활용하여,
다양한 디자인 요소와 형태를 포함하는 새로운 이미지를 대량으로 생성했고, 이렇게 생성된 이미지는
딥러닝 모델의 학습 데이터로 사용되었다.

데이터 라벨링은 모델 학습의 핵심 과정 중 하나로, 이는 모델에게 정답지를 제공하는 역할을 한다.
IMChair의 딥러닝 모델에서는 이미지의 주요 특성과 연관된 형용사로 라벨링을 진행했다. ‘hard’와
‘soft’, ‘static’과 ‘dynamic’ 등의 형용사는 각 이미지의 특성을 설명하는 데 사용되었다. 예를 들어,
dynamic의 이미지를 생성하기 위한 프롬프트로 ‘images with irregular and energetic visual
elements in a dynamic composition’이 사용되었다.

Stable Diffusion 모델은 이미 대량의 데이터로 학습되었기 때문에, 키워드와 연관된 이미지를
생성하는 데 있어 주관적인 의견의 영향을 최소화할 수 있다. 이는 이미지 생성 과정에서 주관적인 해석을
줄이고, 데이터에 기반한 보다 객관적이고 일관된 결과물을 만들어낼 수 있게 한다.

결과적으로 이러한 접근은 IMChair의 딥러닝 모델 구성에서 주관적인 판단이나 편향을 최소화하고,
다양한 디자인 요소와 형태를 탐색하는 데 있어 보다 광범위한 가능성을 제공한다. 이를 통해, 이미지
데이터 수집 과정에서 효율성과 창의성을 향상시킬 수 있었다.
